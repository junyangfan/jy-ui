::: tip 提示
jy-ui 是基于 Vue3 + Ant Design Vue 二次开发的组件库集合
:::


# 快速开始

## 安装
```bash
npm install jy-ui -S
或
yarn add jy-ui -S
```

## 使用
```ts
// main.ts
// 引入组件
import JyUi from 'jy-ui'
// 引入样式
import 'jy-ui/lib/style.css'

// 使用组件
createApp(App).use(JyUi).mount('#app')
```
